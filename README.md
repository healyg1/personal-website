# Personal Website

A minimal static personal website built with HTML, CSS, and JavaScript.

## Files

- `index.html` contains the page content and links.
- `styles.css` contains the visual design.
- `script.js` updates the local-time line.
- `assets/portrait-placeholder.png` is a generated placeholder image you can replace.

## Customize

1. Update `Galvin Healy`, the contact email, and the GitHub profile URL if you want different public details.
2. Update the project titles, descriptions, and repository links.
3. Replace `assets/portrait-placeholder.png` with your own image.

## Publish With GitHub Pages

1. Push this repository to GitHub.
2. In GitHub, open the repository settings.
3. Go to **Pages**.
4. Set the source to the `master` branch and the repository root.
5. Save, then wait for GitHub to publish the site URL.

No build step is required.
